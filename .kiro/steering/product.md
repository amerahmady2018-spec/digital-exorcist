# Product Overview

**The Digital Exorcist** is a file management application with a spooky, supernatural theme. The application helps users manage and organize their files through an engaging, dark-themed interface that balances aesthetic appeal with practical functionality.

## Theme & Tone

- **Spooky but Functional**: Dark mode interface with supernatural theming
- Thematic naming conventions for code (e.g., `banishFile` instead of `deleteFile`)
- Professional comments and documentation despite thematic code
- User experience prioritizes both atmosphere and usability
